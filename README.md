# Aplikacja pogodowa
Projekt aplikacji pokazującej informacje o pogodzie i zanieczyszczeniu powietrza, wykorzystujący API [openWeatherMap](https://openweathermap.org/api).

## Uruchamianie
Aby uruchomic projekt pobierz go na swój komputer, zainstaluj pakiety widoczne w pliku `package.json`, a następnie wykonaj jedno z poleceń uruchamiających, np: `npm run dev`.
Upewnij się, że masz zainstalowane [node.js](https://nodejs.org/en).
